# Ratio demo app (applicant test task)

This repo is a **demo app for a test task for applicants**.

## Task

Go to the repo **Issues** and read the issue about **data table view**.

## Repo structure

- **Monorepo**: Turbo Repo
- **Package manager / runtime**: Bun
- **App**: `@ratio/demo`

## Getting started

Install dependencies:

```bash
bun install
```

Run dev:

```bash
bun dev
```
